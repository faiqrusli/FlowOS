## Implementation status note

Several SRS future enhancements have since been partially or fully implemented (e.g. Notes module, weekly reflection, focus tracking). Others remain on the long-term roadmap (Goals, AI Coach, mobile apps, calendar sync). The future enhancements document includes a **2026 status** column where applicable — the original SRS text is preserved unchanged above that summary.
