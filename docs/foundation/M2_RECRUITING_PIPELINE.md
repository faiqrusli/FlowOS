## Outreach template

Personalize the first line. Send from your real email — not a noreply.

```
Subject: Trying a daily productivity experiment (10 min?)

Hi [Name],

I'm building FlowOS — one screen for today's tasks, habits, focus, and reflection.
I use it daily myself at https://flowos-sage.vercel.app and I'm looking for [3–5] people
who already juggle 2+ productivity apps to try it for two weeks.

No pitch deck — I'd watch you use it for 10 minutes on a quick call, then check in on day 3 and day 7.
Desktop web only for now; single-device-primary (honest limitation).

Worth a short call this week or next?

— [Founder name]
```

---

## Day-0 onboarding call script (outline)

1. **Minutes 0–10:** Founder says little — watch screen share. Note:
   - First navigation choice
   - First task action
   - First focus attempt
   - First confusion / stall
2. **Minutes 10–15:** Answer questions only.
3. Set **day-3 async check-in** (send feedback template below).
4. State limitations plainly: desktop web only; single-device-primary until a fix ships.
5. Do **not** pitch roadmap or defend design — observe.

---

## Async feedback template (day 3 / day 7)

Send after the call or on check-in day:

```
Quick FlowOS check-in — 5 questions (reply in bullet points):

1. What did you try to do first when you opened it?
2. Where did you stall or get confused?
3. What did you use outside FlowOS because FlowOS failed?
4. Would you open this tomorrow unprompted? Why / why not?
5. One thing to delete; one thing to keep.
```
