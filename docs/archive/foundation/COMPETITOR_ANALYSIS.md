# Competitor Analysis

**Status:** Active  
**Audience:** Founder, product  
**Last updated:** July 3, 2026  
**Rule:** Learn from competitors; do not copy features without loop value.

FlowOS today: **5.5/10 product** — feature breadth without daily loop coherence ([archive/foundation/CEO_REVIEW_JULY_2026.md](../archive/foundation/CEO_REVIEW_JULY_2026.md)).

---

## Category map

| Category | Products | FlowOS relationship |
|----------|----------|---------------------|
| Task managers | Todoist, TickTick, Things 3 | Forethought module; FlowOS adds focus + reflection |
| Daily planners | Sunsama, Akiflow, Routine | **Closest competitors** |
| AI schedulers | Motion | Different wedge; FlowOS defers AI |
| Knowledge bases | Notion, Obsidian | Notes module only; not primary fight |
| Team tools | Linear | Non-goal; ignore |

---

## Todoist

| | |
|---|---|
| **Strengths** | Best-in-class capture, natural language, cross-platform, trusted brand |
| **Weaknesses** | No native focus timer; weak reflection; habits require integrations |
| **Learn** | Speed of capture; inbox zero mental model |
| **Differ** | Do not chase NLP capture before inline capture works; win on **loop** not input tricks |

## TickTick

| | |
|---|---|
| **Strengths** | Tasks + habits + pomodoro in one app; calendar views; affordable |
| **Weaknesses** | Cluttered UI; weak reflection; habit gamification feels generic |

| **Differ** | Calmer dark UI; reflection as first-class, not checkbox; no gamification |

| **Weaknesses** | Apple-only; no habits/focus/reflection integration; no web |

## Sunsama

| | |
|---|---|
| **Strengths** | Daily planning ritual; calendar + task pull; guided reflection; clear positioning |
| **Weaknesses** | $20/mo; calendar-dependent; heavy for casual users |
| **Learn** | **Daily planning ceremony**; plan → work → shutdown flow |
| **Differ** | No calendar sync required; lower price; habits + focus native; indie privacy |

**Highest strategic threat** in daily-planner category.

## Akiflow

| | |

| **Weaknesses** | Complex setup; calendar-centric; expensive |
| **Learn** | Command palette / keyboard power user path |

## Routine

| | |

| **Weaknesses** | Smaller user base; platform limits |

## Motion

| | |
|---|---|
| **Strengths** | AI auto-scheduling; team features; aggressive growth |
| **Weaknesses** | AI dependency; trust issues; expensive; over-scheduling fatigue |

---

## Where FlowOS should compete (12 months)

1. **Integrated loop** — plan + focus + reflect without four apps  
2. **Calm dark workspace** — lamplit desk vs bright dashboards  
3. **Indie price / privacy** — below Sunsama; no AI upsell  
4. **Habit + focus depth** — deeper than Sunsama on execution tracking  

---

## Where FlowOS should not compete

1. Calendar replacement (FE-11 deferred)  
2. AI auto-scheduling (Motion's game)  
3. Team workflows (Linear's game)  
4. Knowledge management (Notion/Obsidian's game)  
5. Mobile-first capture (Todoist's moat)
